# Defa Store App

تطبيق متجر **دِفــــا**  
حيث يبدأ الذوق الرفيع ✨

تطبيق Flutter مخصص لعرض وبيع المنتجات الراقية:
- الإكسسوارات
- الهدايا
- المباخر
- خواتم العقيق
- التحف والمنحوتات

## المميزات
- واجهة فاخرة داعمة للعربية (RTL)
- ربط مباشر مع Firebase
- عرض ذكي للصور (Smart Media)
- نظام عملات ديناميكي
- تجربة استخدام راقية وبسيطة

## التقنيات المستخدمة
- Flutter
- Firebase (Auth, Firestore, Storage)
- Cloudinary (Media Processing)
- Material 3
